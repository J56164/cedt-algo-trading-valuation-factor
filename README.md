# CEDT Algo trading - Valuation Factor

## Installation

Install [uv](https://docs.astral.sh/uv/), then run:

```sh
uv sync
```

To run:

- Navigate to a notebook
- If you're in VSCode, Ctrl+Shift+P -> Python: Select Interpreter
- Run the notebook in VSCode
